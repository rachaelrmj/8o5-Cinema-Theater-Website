# 805 Cinema

805 Cinema is a responsive single-page web application that serves as a digital movie theater experience. Users can browse currently playing films, search showtimes, watch trailers, explore upcoming releases, and interact with theater membership features through a dynamic and engaging interface.

The project demonstrates front-end web development concepts including asynchronous data retrieval, client-side account management, responsive design, JSON data processing, and third-party plugin integration.

---

## Project Overview

805 Cinema was designed to recreate the experience of visiting a modern movie theater website while showcasing interactive web technologies and user-focused design principles.

Visitors can:

* Browse currently playing movies
* Search showtimes by movie and date
* Watch movie trailers
* Explore upcoming releases
* Learn about featured dining options
* Join the theater loyalty program
* Manage membership information through browser storage

The application uses a local JSON file as a simulated backend database, allowing dynamic content generation without requiring a server-side environment.

## Features

### Now Playing Showcase

* Animated movie poster slider
* Highlights currently playing films
* Continuous CSS-powered animation

### Showtime Search

* Search movies by title and date
* Dynamically displays available showtimes
* Retrieves data from a local JSON database

### Trailer Viewer

* Watch movie trailers directly from search results
* Dynamic content loading with JavaScript

### New Releases Carousel

* Interactive rotating movie carousel
* Showcases upcoming film releases
* Powered by the JasCarousel jQuery plugin

### 8o5 Club Membership

* Create and manage theater club accounts
* Client-side login and account validation
* Password recovery functionality
* Browser-based account persistence using Local Storage

### Featured Dining Experience

* Interactive weekly "Munchie Deal"
* jQuery UI dialog integration
* Enhanced user engagement through dynamic content
